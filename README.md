# Homebrew Tap for Kon

Homebrew formulae for [Kon](https://github.com/Kaden-Schutt/kon) — let Claude reach your computer.

## Install

```bash
brew tap schuttdev/tap https://github.com/Kaden-Schutt/homebrew-tap
brew install kond
```

The client is installed separately (typically by Claude inside a sandbox):

```bash
brew install kon
```

## Upgrade

```bash
brew update
brew upgrade kond
```
